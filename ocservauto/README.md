##OCSERV-AUTO
这是一枚适用于deibian_7的openconnect_server安装脚本。

============

##USAGE
```shell
apt-get update
apt-get install wget
wget http://git.io/p9r8 --no-check-certificate -O ocservauto.sh
bash ocservauto.sh
```
